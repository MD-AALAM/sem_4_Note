Getter:
->getters in python are methods that are used to access the values of an
ojects properties.they are used to return the value of a specific property,
and are typically defined using the @propety decorator.

Setter:
->it is important note that the getters do not take any parameters and we 
cannot set the value through getter method.for that we need setter method 
which can be added by decorating method with @property_name.setter
