🔷 Software Interrupts and Exceptions
🔹 Introduction

An interrupt is a signal that temporarily stops the normal execution of a program and transfers control to a special routine called the Interrupt Service Routine (ISR).

👉 It tells the CPU:
“Stop current task and handle this event.”

🔹 Types of Interrupts
Hardware Interrupts → Generated by hardware devices
Software Interrupts → Generated by program instructions

🔹 Software Interrupt
A software interrupt is generated by executing a special instruction in a program.
It is invoked by software itself.

🔸 Types of software interrupts:
1.Normal Software Interrupt
Generated intentionally by program
Example: system calls
2.Exception
Unplanned interrupt due to error during execution
🔸 Examples:
Divide by zero
Invalid instruction
Arithmetic overflow

👉 In 8085 microprocessor, there are interrupts like:
RST0 to RST7
