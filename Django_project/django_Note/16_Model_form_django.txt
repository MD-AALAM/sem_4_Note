🔷 Model Forms in Django
🔶 What is a Model Form?

👉 A ModelForm is a form that is directly connected to a Model.

📌 It automatically creates form fields from model fields

🔷 Simple Definition (Exam Point)

ModelForm is a Django form that is automatically generated from a model and used to create/update database records.

🔷 1. Why ModelForm?

Without ModelForm:
❌ Write form fields manually
❌ Write save logic manually

With ModelForm:
✔ Less code
✔ Automatic validation
✔ Direct database saving

🔷 2. Create Model (Example)

👉 In models.py:

from django.db import models

class Student(models.Model):
    name = models.CharField(max_length=100)
    age = models.IntegerField()

    def __str__(self):
        return self.name

🔷 3. Create ModelForm

👉 In forms.py:

from django import forms
from .models import Student

class StudentForm(forms.ModelForm):
    class Meta:
        model = Student
        fields = ['name', 'age']

🔶 Explanation:
model → Which model to use
fields → Which fields to include
🔷 4. Use ModelForm in View
from django.shortcuts import render
from .forms import StudentForm

def student_form(request):
    form = StudentForm()

    if request.method == 'POST':
        form = StudentForm(request.POST)
        if form.is_valid():
            form.save()   # Saves data to database

    return render(request, 'form.html', {'form': form})

🔷 5. Display in Template
<form method="POST">
    {% csrf_token %}
    {{ form.as_p }}
    <button type="submit">Submit</button>
</form>

🔷 6. How ModelForm Works (Important 🔥)
Model → ModelForm → Form Fields → User Input → Save → Database


👉 Automatically:
✔ Creates input fields
✔ Validates data
✔ Saves to database

🔷 7. Customizing ModelForm
🔸 Add Labels
class StudentForm(forms.ModelForm):
    class Meta:
        model = Student
        fields = ['name', 'age']
        labels = {
            'name': 'Student Name',
        }

🔸 Add Widgets (Design)
widgets = {
    'name': forms.TextInput(attrs={'class': 'form-control'}),
}

🔷 8. Update Existing Data
def update_student(request, id):
    student = Student.objects.get(id=id)
    form = StudentForm(instance=student)

    if request.method == 'POST':
        form = StudentForm(request.POST, instance=student)
        if form.is_valid():
            form.save()

    return render(request, 'form.html', {'form': form})

🔷 10. Common Mistakes

❌ Not using instance for update
❌ Missing csrf_token
