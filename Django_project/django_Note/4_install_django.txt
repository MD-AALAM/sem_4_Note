🔷 Install Django (Step-by-Step)
🔶 1. Check Python Installation

Django works on Python, so first check:

python --version


👉 Output should be something like:

Python 3.x.x


❌ If not installed → download Python from official website.

🔶 2. Create Virtual Environment (Recommended ⭐)

👉 This keeps your project clean and professional.

📌 Command:
python -m venv env

🔶 3. Activate Virtual Environment
👉 For Windows:
env\Scripts\activate


👉 After activation, you will see:

(env) C:\Users\...

🔶 4. Install Django
📌 Command:
pip install django

🔶 5. Verify Installation
django-admin --version


👉 Example output:

5.x.x


✅ Means Django installed successfully 🎉

🔷 Important Commands Summary
python -m venv env
env\Scripts\activate
pip install django
django-admin --version

🔷 Common Errors & Fix
❌ Error: pip not recognized

👉 Solution:

python -m pip install django

❌ Error: virtual environment not activating

👉 Use:

Set-ExecutionPolicy Unrestricted


(only in PowerShell)

🔷 Pro Tip (Very Important)

👉 Always install Django inside virtual environment

✔ Prevents version conflicts
✔ Used in real companies
✔ Best practice for projects
