🔷 Admin Panel in Django
🔶 What is Admin Panel?

👉 Django provides a built-in Admin Interface.

📌 It allows you to manage your database without writing code

🔷 Simple Definition (Exam Point)

Admin Panel is a built-in feature in Django used to manage models and database data through a web interface.

🔷 1. Why Admin Panel?

Without Admin Panel:
❌ Need to write SQL queries
❌ Hard to manage data

With Admin Panel:
✔ Add data easily
✔ Edit data
✔ Delete data
✔ Manage users

🔷 2. Steps to Use Admin Panel
🔶 Step 1: Create Superuser
python manage.py createsuperuser


👉 Enter:

Username
Email
Password
🔶 Step 2: Run Server
python manage.py runserver

🔶 Step 3: Open Admin Panel

👉 Open browser:

http://127.0.0.1:8000/admin/


👉 Login using superuser credentials

🔷 3. Register Model in Admin

👉 By default, models are not visible in admin panel.

🔶 In admin.py:
from django.contrib import admin
from .models import Student

admin.site.register(Student)

🔶 Result:

👉 Now you can:

Add Student
Edit Student
Delete Student
🔷 4. Customize Admin Panel (Important 🔥)
🔸 Display Fields
class StudentAdmin(admin.ModelAdmin):
    list_display = ('name', 'age')

admin.site.register(Student, StudentAdmin)

🔸 Add Search
class StudentAdmin(admin.ModelAdmin):
    search_fields = ('name',)

🔸 Add Filters
class StudentAdmin(admin.ModelAdmin):
    list_filter = ('age',)

🔷 5. Improve Model Display

👉 In models.py:

class Student(models.Model):
    name = models.CharField(max_length=100)

    def __str__(self):
        return self.name


👉 Makes admin panel cleaner
