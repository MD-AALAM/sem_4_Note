📘 DJANGO FULL NOTES (Beginner → Advanced)
🔷 1. What is Django?

Django is a high-level Python web framework used to build secure and scalable websites quickly.

👉 Example:

Instagram (uses Django)
E-commerce websites
Blog systems
🔷 2. Why Django?

✔ Fast development
✔ Built-in security
✔ Uses Python (easy language)
✔ Comes with admin panel
✔ Follows clean architecture

🔷 3. Django Architecture (MVT)

Django follows MVT pattern:

Component	Meaning	Work
Model	Database	Stores data
View	Logic	Handles request
Template	UI	HTML display

👉 Flow:
Client → URL → View → Model → Template → Response

🔷 4. Install Django
pip install django


Check version:

django-admin --version

🔷 5. Create Project
django-admin startproject myproject
cd myproject
python manage.py runserver


👉 Open browser:

http://127.0.0.1:8000/

🔷 12. Models (Database)

👉 models.py:

from django.db import models

    name = models.CharField(max_length=100)
    age = models.IntegerField()

🔷 15. Forms
from django import forms

class StudentForm(forms.Form):
    name = forms.CharField()

🔷 16. Model Forms
from django.forms import ModelForm
from .models import Student

class StudentForm(ModelForm):
    class Meta:
        model = Student
        fields = '__all__'

🔷 17. GET vs POST
Method	Use
GET	Fetch data
POST	Send data
if request.method == 'POST':
    name = request.POST['name']

🔷 18. CRUD Operations
Create
Student.objects.create(name="Ali", age=20)

Read
Student.objects.all()

Update
obj = Student.objects.get(id=1)
obj.name = "Majid"
obj.save()

Delete
obj.delete()

🔷 19. Django ORM

👉 No SQL needed:

Student.objects.filter(age=20)

🔷 20. Relationships
Foreign Key (Many to One)
class Course(models.Model):
    name = models.CharField(max_length=100)

class Student(models.Model):
    course = models.ForeignKey(Course, on_delete=models.CASCADE)
