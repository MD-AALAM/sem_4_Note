🔷 CRUD Operations in Django
🔶 What is CRUD?

👉 CRUD stands for:

📌 Create, Read, Update, Delete

👉 These are the basic operations performed on database

🔷 Simple Definition (Exam Point)

CRUD operations are used to create, retrieve, update, and delete data in a database using Django models.

🔷 1. Create (Insert Data)

👉 Used to add new data

🔶 Example:
from .models import Student

def create_student(request):
    Student.objects.create(name="Majid", age=20)
    return HttpResponse("Student Created")

🔶 Using ModelForm (Best Practice ⭐)
if form.is_valid():
    form.save()

🔷 2. Read (Retrieve Data)

👉 Used to get data from database

🔶 Example:
def show_students(request):
    students = Student.objects.all()
    return render(request, 'home.html', {'students': students})

🔶 Template:
{% for student in students %}
    <p>{{ student.name }} - {{ student.age }}</p>
{% endfor %}

🔷 3. Update (Modify Data)

👉 Used to change existing data

🔶 Example:
def update_student(request, id):
    student = Student.objects.get(id=id)
    student.name = "Ali"
    student.save()
    return HttpResponse("Updated")

🔶 Using ModelForm:
student = Student.objects.get(id=id)
form = StudentForm(request.POST, instance=student)

if form.is_valid():
    form.save()

🔷 4. Delete (Remove Data)

👉 Used to delete data

🔶 Example:
def delete_student(request, id):
    student = Student.objects.get(id=id)
    student.delete()
    return HttpResponse("Deleted")

🔷 5. Full CRUD Flow
Create → Insert data
Read   → Display data
Update → Modify data
Delete → Remove data

🔷 6. Real-Life Analogy

Think CRUD as:

📚 Library System

Create → Add new book
Read → View books
Update → Edit book details
Delete → Remove book
