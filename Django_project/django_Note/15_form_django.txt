🔷 Forms in Django
🔶 What is a Form?

👉 A Form is used to take input from the user.

📌 Examples:
Login form
Registration form
Contact form
🔷 Simple Definition (Exam Point)

Form is used to collect and validate user input in Django.

🔷 1. Why Forms?

Without forms:
❌ No user input
❌ No interaction

With forms:
✔ Take user data
✔ Validate data
✔ Send data to server

🔷 2. Types of Forms in Django
🔸 (A) HTML Form (Basic)

👉 Simple form using HTML

<form method="POST">
    <input type="text" name="username">
    <button type="submit">Submit</button>
</form>

🔸 (B) Django Form (Recommended ⭐)

👉 Built using Django

from django import forms

class StudentForm(forms.Form):
    name = forms.CharField(max_length=100)
    age = forms.IntegerField()

🔷 3. Handle Form in View
from django.shortcuts import render
from .forms import StudentForm

def form_view(request):
    form = StudentForm()

    if request.method == 'POST':
        form = StudentForm(request.POST)
        if form.is_valid():
            name = form.cleaned_data['name']
            age = form.cleaned_data['age']

    return render(request, 'form.html', {'form': form})

🔷 4. Display Form in Template
<form method="POST">
    {% csrf_token %}
    {{ form.as_p }}
    <button type="submit">Submit</button>
</form>

🔶 Important:

👉 {% csrf_token %} is required for security

🔷 5. Model Form (Very Important 🔥)

👉 Automatically connects form with model

Example:
from django import forms
from .models import Student

class StudentForm(forms.ModelForm):
    class Meta:
        model = Student
        fields = ['name', 'age']

View:
def form_view(request):
    form = StudentForm()

    if request.method == 'POST':
        form = StudentForm(request.POST)
        if form.is_valid():
            form.save()

    return render(request, 'form.html', {'form': form})

🔷 6. GET vs POST
🔸 GET
Data visible in URL
Used for fetching data
🔸 POST
Data hidden
Used for submitting data

👉 Forms usually use POST

🔷 7. Form Validation
